## How It Works

1. The report HTML includes a container: `<div id="qrCode"></div>` positioned at bottom right
2. When the report window loads, `generateReportQRCode()` is called
3. The function generates a QR code that links to the verification page
4. The QR code is displayed at the bottom right of each report with border
